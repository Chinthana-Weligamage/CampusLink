# Demo Script

1. Start all services and the gateway.
2. Open the gateway route index and Swagger URLs.
3. Register a student account.
4. Log in and capture the JWT.
5. Connect to the notification SSE stream.
6. Book a shuttle seat and show the booking notification.
7. Create an assignment as admin.
8. Submit work as a student and show the submission notification.
9. Fetch notifications and mark one as read.
